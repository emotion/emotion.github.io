# 树莓派使用记录

1. 安装

首先用读卡器在系统上，给闪迪的32G tf卡，用Raspberry Pi Imager下载烧录系统。我这里使用的是ubuntu的64bit lts版。

第一次进入系统，用root更新系统出现问题，怀疑是由于ubuntu的每日自动更新和手动更新冲突，导致的问题，因为我重新烧录后，看到每日更新跑了很久。

使用Raspberry Pi Imager不会将tf卡已烧录的系统擦除掉，这样会导致只能使用第一次烧录剩下的空间，导致空间不足出现各种问题。

可以使用macos的diskutil将数据完全擦除

```
# 查看并查找对应的disk
diskutil list
# 擦除disk的所有数据，包括分区
sudo diskutil eraseDisk ExFAT ud /dev/disk3 
```

2. 移动硬盘自动挂载

```
# 列出所有的disk uuid
blkid
# 修改/etc/fstab 加上这个
UUID=8E0E912D0E910F79   /mnt/MyPassport ntfs-3g default,nofail,noexec,umask=0000        0       0
```

3. 修改用户名和hostname

```
# 将用户名从默认的ubuntu修改为自定义
usermod -l newname oldname
mv /home/oldname /home/newname
usermod -d /home/newname -m newname
# 修改hostname
hostname newHostName
# hostname RasPi
```

8. 散热问题

安装lm-sensors，并用sensors扫描并查看温度

或者直接查看/sys/class/thermal/thermal_zone0/temp，这个值是摄氏度*1000

重新对之前PrivateHD下载的资源做种，会进行数据校验，rtorrent cpu会跑到80%左右，ntfs-3g跑到40%，然后cpu温度一度跑到差不多80度，但是cpu总体利用率才15%。所以被动散热还是有问题，如果较长时间cpu负荷较高，可能把机器跑挂。

即使有金属外壳用来散热，但是散热效果还是不理想（虽然比单独塑料壳略好），还是需要小风扇进行散热。金属壳+小风扇能把温度压在50度左右
